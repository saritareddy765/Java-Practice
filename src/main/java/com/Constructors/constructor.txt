Constructor:
            * A constructor is a block of codes similar to the method
            * It is called when an instance (object) of the class is created.
            * Every time an object is created using the new keyword, at least one constructor is called.
            * constructor don't have a return type.

Types:
            * Default Constructor.
            * Parameterized Constructor.

Default Constructor:
            --> A constructor with no parameters.
            --> Syntax :
                         public ClassName()
                         {

                          }

Parameterized Constructor:
            --> A constructor that accepts parameters to initialize object attributes.
            --> Syntax:
                         public ClassName(int a)
                         {

                         }
