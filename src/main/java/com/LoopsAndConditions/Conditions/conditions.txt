Conditions:
           * Conditional statements control the flow of execution
                 based on a condition's evaluation (true/false).

Types:
           * if condition
           * if-else condition
           * else-if condition
           * switchCase

if condition:
           * if Statement Executes a block of code if the condition is true.
           * Syntax:
                    --> if(condition)
                           {
                           // block of code (true executed)
                           }

if-else condition:
           * if-else Statement Executes one block of code if the condition is true;
                otherwise, executes another block.
           * Syntax:
                    --> if(condition){
                            // block of code (true executed or go to else)
                         }else{
                            // block of code
                         }

else-if condition:
           * else if Ladder Checks multiple conditions sequentially.
           * Syntax:
                   --> if(condition1){
                           // block of code (true executed or go to the condition2)
                        }else-if(condition2)
                        {
                           // block of code (true executed or go to else)
                        }else{
                          // block of code
                        }
