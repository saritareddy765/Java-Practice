thisKeyword:
            * The this keyword is used to differentiate between
                   instance variables and parameters when they have the same name.
            * It is pointing to the current class object.
            * It is used to refer current class instance variable.
            * thisKeyword can be used to invoke current class method(implicitly).
            * this() can be used to invoke current class constructor.
constructor chaining:
            * calling one constructor inside another constructor of the same class.
